### 7.1.2 定义类

#### 定义类（struct类）

* 成员函数：声明（必须）在类内部，定义（一般）在类外部
  * 定义在类内部的成员函数是隐式的inline函数
* 作为借口的非成员函数：声明、定义在类外部

#### 引入this

* this：是用该类定义的对象的隐式const指针，类型为 类 *const this
* 可以通过this->来访问外类类型对象内部的元素
* 也可以不加。。。这个时候编辑器隐式访问this

```C++
// 效果相同
return this->bookNo;
return bookNo;
```

#### this设置为底层const

* 方法：在参数列表的`)`后面加上const
* 学名：常量成员函数
* 常量对象、常量对象的引用或指针都只能调用常量成员函数

#### 类作用域和成员函数

* 类本身是一个作用域（命名空间）
* 类内的所有成员/属性都可以相互访问
  * 原理：编译器首先编译成员声明，再编译函数体

#### 在类外部定义成员函数

* 使用作用域运算符来说明这是在哪个类内
* 外部定义可以使用内部属性和函数

【示例】

```C++
double Sales_data::avg_price()const{
    ...
}
```

#### 定义一个范围this对象的函数

```C++
Sales_data& Sales_data::combine(){
    return *this;
}
```
