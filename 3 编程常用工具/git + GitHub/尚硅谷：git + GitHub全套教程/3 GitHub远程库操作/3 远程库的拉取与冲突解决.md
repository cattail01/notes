### 远程库的拉取

#### git fetch [远程库名字] [远程库分支名]：抓取操作，将远程内容下载到本地

* 但是不会修改工作区内容，只是创建了一个远程库下载后的分支
* 解决方法：git checkout [远程库别名/远程库分支名]
* git merge [远程库别名/远程库分支名]：将远程库的内容合并到本地库

#### git pull [远程库名字] [远程库分支名]：相当于 fetch + merge

* 适用于比较简单、不会冲突的方式

### 冲突解决

#### 冲突后发生：

* 发生冲突时，只有第一个人能够推送（push）到远程库
* 第二个人需要先拉取（pull）到本地库，合并（merge），然后推送到给到远程库。直接推送是推送不上去的。
